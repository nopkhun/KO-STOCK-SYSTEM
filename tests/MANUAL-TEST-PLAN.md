# คู่มือทดสอบด้วยมือ — KO-Stock-System

ใช้ checklist นี้ทดสอบฟีเจอร์ **นำเข้า, โอนย้าย, เบิกออก, นับสต็อก และการแสดงผล Dashboard** ให้ถูกต้อง 100%

---

## 1. Dashboard (ยอดสต็อกคงเหลือ)

| # | ขั้นตอน | ผลที่คาดหวัง | ผ่าน ☐ |
|---|--------|--------------|--------|
| 1.1 | เปิดแอป → เลือกสาขา | แดชบอร์ดแสดงการ์ด "สินค้าใกล้หมด", "รายการเคลื่อนไหววันนี้" และตาราง "ยอดสต็อกคงเหลือ" | ☐ |
| 1.2 | ตรวจตารางยอดคงเหลือ | แต่ละรายการแสดง ชื่อสินค้า, ยอดคงเหลือ (จำนวน+หน่วย), สถานะ, ปุ่มโอน/เบิก | ☐ |
| 1.3 | เลือก "แสดงเฉพาะใกล้หมด" | แสดงเฉพาะสินค้าที่ยอดต่ำกว่าขั้นต่ำ (สถานะเติมของ) | ☐ |
| 1.4 | เปลี่ยนเรียงตาม (ชื่อ / ยอดคงเหลือ / ใกล้หมดก่อน) | ลำดับในตารางเปลี่ยนตามที่เลือก | ☐ |
| 1.5 | ค้นหาสินค้า | แสดงเฉพาะรายการที่ตรงกับคำค้น | ☐ |
| 1.6 | กดปุ่ม "รีเฟรช" | ข้อมูลโหลดใหม่จากชีต (ยอดตรงกับที่บันทึกล่าสุด) | ☐ |

| 2.5 | ไปที่ "ประวัติเข้า-ออก" หรือ Audit Log | มีรายการ "นำเข้า" ตามที่บันทึก (จำนวน, สาขา, ผู้ทำ) | ☐ |

| 3.5 | เปลี่ยนไปสาขาปลายทาง → ดูยอดสต็อกคงเหลือ | **ยอดสินค้านั้นเพิ่มตามจำนวนที่โอน** | ☐ |
| 3.6 | ดูประวัติ | มีรายการ "โอน" พร้อมจากสาขา → ไปสาขา | ☐ |

## 4. เบิกออก (Out / นำออก)

| # | ขั้นตอน | ผลที่คาดหวัง | ผ่าน ☐ |
|---|--------|--------------|--------|
| 4.1 | เลือกสาขา → กด "เบิกของ" (หรือปุ่มเบิกที่รายการสินค้า) | เปิด modal เบิกออก | ☐ |
| 4.2 | เลือกสินค้าที่มีสต็อก → จำนวน (ไม่เกินยอดคงเหลือ) → เหตุผลเบิก | กรอกได้ | ☐ |
| 4.3 | กดบันทึก | บันทึกสำเร็จ, modal ปิด | ☐ |
| 4.4 | ดูยอดสต็อกคงเหลือ (Dashboard) | **ยอดคงเหลือของสินค้านั้นลดลงตามจำนวนที่เบิก** (เช่น เบิก 5 จาก 17 → เหลือ 12) | ☐ |
| 4.5 | ดูประวัติ / Audit Log | มีรายการ "นำออก" จำนวนและผู้ทำถูกต้อง | ☐ |

## 5. ตรวจนับสต็อกจริง (Stock Take)

| # | ขั้นตอน | ผลที่คาดหวัง | ผ่าน ☐ |
|---|--------|--------------|--------|
| 5.1 | ไปแท็บ "ตรวจนับสต็อกจริง" | แสดงรายการสินค้าและช่องกรอกจำนวนนับจริง | ☐ |
| 5.2 | กรอกจำนวนนับจริง (ต่างจากยอดระบบ) สำหรับบางรายการ | ค้นหา/กรอกได้ | ☐ |
| 5.3 | กด "บันทึกและปรับยอด" → ยืนยัน | บันทึกสำเร็จ | ☐ |
| 5.4 | กลับไป Dashboard / ยอดสต็อกคงเหลือ | **ยอดคงเหลือเป็นตามจำนวนนับจริงที่กรอก** (ปรับยอดแล้ว) | ☐ |
| 5.5 | ดูประวัติ | มีรายการนำเข้า/นำออก (ปรับยอดนับจริง) ตามผลต่าง | ☐ |

---

## 6. สรุปความถูกต้อง 100%

- **Dashboard:** แสดงยอดจาก Inventory ล่าสุด (หลังนำเข้า/โอน/เบิก/นับสต็อก)  
- **นำเข้า:** ยอดเพิ่ม + มีในประวัติ  
- **โอน:** ยอดต้นทางลด ยอดปลายทางเพิ่ม + มีในประวัติ  
- **เบิกออก:** ยอดลด + มีในประวัติ  
- **นับสต็อกจริง:** ยอดเป็นตามจำนวนนับจริง + มีในประวัติ  

ถ้า checklist ด้านบนผ่านทั้งหมด ถือว่าฟีเจอร์นำเข้า โอน เบิก นับสต็อก และการแสดงผล Dashboard ทำงานถูกต้อง 100% ตามที่ออกแบบ
